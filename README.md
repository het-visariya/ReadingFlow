## 🌟 Overview

ReadingFlow is a next-generation digital library browsing experience developed during my internship to modernize the way users discover and explore books.

Instead of presenting users with a traditional list of books, ReadingFlow introduces an interactive, animation-driven interface where users can navigate through book collections using immersive transitions, smooth animations, and visually engaging browsing experiences.

The goal of the project is to bridge the gap between conventional library management systems and modern user experiences inspired by premium product websites.

---

# 🎯 Why I Built ReadingFlow

Traditional library systems generally suffer from:

- Static interfaces
- Poor user engagement
- Difficult book discovery
- Outdated browsing experiences
- Low interaction quality
- Limited visual feedback

ReadingFlow was built to solve these problems by creating an interactive reading experience that encourages exploration while keeping the interface clean, modern, and intuitive.

## 💡 Real World Problem Solved

ReadingFlow demonstrates how a traditional library catalogue can be transformed into an engaging digital experience.

Instead of users endlessly scrolling through long tables of books, they can visually discover books in an intuitive and interactive way.

This approach increases:

- Overall user satisfaction

# ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/het-visariya/ReadingFlow.git
```

Move into the project

```bash
cd ReadingFlow
```

Install dependencies

```bash
npm install
```

---

# ▶️ Run the Development Server

```bash
npm run dev
```

The application will start locally.

Usually at

```
http://localhost:5173
```
